SERIKA AI TOOLKIT是一个由Python编写的AI工具箱，集合了翻译、扩写、润色、听力四种功能，通过大语言模型执行任务。
注意：执行任务时请选择恰当的模型，模型可能不会遵循你设置的部分选项。你可以添加多个模型（注意！API key明文存储在llm文件夹下，谨防泄露）
-
①翻译功能：
自定义目标语言：选择自定义语言后，你可以填入语种名称和国家或地区。
由于是AI提示词形式所以它的填写很灵活。选择国家或地区的原因是由于不同国家或地区的语言用词习惯不同，如西班牙语（西班牙和拉美）、英语（英式英语和美式英语，尤其是翻译楼层时）、简体中文（中国大陆和新加坡）、繁体中文（中国香港/澳门、中国台湾）。他们的用词可能有区别因此需要进行区分。
保留通用简称：保留AI、MCP等等常用的英文简称，他们将不予翻译
自动纠错：勾选后模型会尝试自动选择符合语境的用词，防止乱翻
格式保留：换行或标点符号保留
政治遵守：要求AI遵循输入者所在区域的法律法规（如钓鱼岛严禁翻译为尖阁诸岛，无论中文还是日语汉字）

②扩写功能：
可以选择字数和容差（字数和容差计算不包含标点符号）。
部分AI对字数不敏感，因此一旦字数与要求偏离度超过容差则软件自动打回重写，超过最大打回次数则自动停止扩写。
下方可以选择是否契合原文以及经典的政治遵循按钮，也可以填写性质

③润色功能：
目标风格：如果没有也可以自定义写一个风格或风格要求
错词纠正：AI会主动帮你纠正错误的词汇，如果你的文章中有故意写错的词汇则需要关闭它命令模型保留错误
用辞纠正：填入对应的国家或地区，模型会自动依据当地语言习惯进行转译。注意这可能会影响润色后的输出语言。
政治遵循：依旧防止模型夹带私货

④听力功能：
上传音频，系统会自动转换为mp3上传。
可能出现的语言：需要以文本形式填写可能出现的语言防止识别错误（例如将西语音频当成英语识别），格式自由因为这是提示词类别。
国家或地区：依旧是因为各地区语言习惯不同，如西班牙和拉美。
注意！请确保你的服务商支持音频端点，以及你的模型是多模态，否则会出错。

全局设置：设置较少，主要是主题颜色设置和是否开启崩溃显示
模型设置：用于填写你的AI API，可以添加多个模型，添加的每个模型配置会以json文件形式存储在llm文件夹下。
消耗记录：查看软件消耗了多少token
